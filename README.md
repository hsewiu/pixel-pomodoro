# 一個Lofi風格的番茄鐘

- 主要是：Lo-fi 風格 (?) 番茄鐘工具
- Avatar：有代表自己的小人物跟大家一起走來走去工作，設置個性簽名等等，一起倒數計時
- 音樂功能：目前只支援我選定的幾部任天堂遊戲音樂
