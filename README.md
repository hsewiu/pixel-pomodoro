# 🍅 Pixel Pomodoro World - 多人線上像素專注平台

這是一個結合了 Lo-fi 風格、多人即時互動與番茄鐘工作法的線上專注平台。在專注的同時，你不再是孤軍奮戰，而是與來自全球的夥伴一起在像素世界中努力。

## 🎧 使用看看
前往： https://hsewiu.github.io/pixel-pomodoro/

## ✨ 核心特色
**1. 多人即時共學 (Real-time Presence)**
- 視覺化社交空間：在底部走廊，你可以看到所有正在線上專注的夥伴，每個人的位置與動作都是即時同步的。
- 個性化角色：提供多種可愛的emoji頭像與自定義名稱，展現你的個人風格。
- 即時狀態更新：除了番茄鐘剩餘時間，你還可以設置個人簽名（Bio），讓大家知道你當前的想法。

**2. 沉浸式專注體驗**
- 經典番茄鐘：內建 25, 30, 45, 60 分鐘等常用時段，並配備完成提示音。
- Lo-Fi 音樂電台：整合多款經典遊戲風格（薩爾達、薩爾達、卡比等）的 Lo-Fi 音頻流，點擊即播，為你營造完美的專注氛圍。
- CRT 復古濾鏡：獨特的 CRT 掃描線效果與發光視覺設計，帶你重回 8-bit 的懷舊時代。

**3. 個人化設定**
- 深色/明亮模式：隨時切換主題色調，無論是深夜衝刺還是午後閱讀都能擁有舒適的視覺體驗。
- 線上人數統計：左上角顯眼的發光標籤，讓你隨時感受到社群的陪伴。

## 🛠️ 技術架構
前端框架: React.js (Vite)
樣式處理: Tailwind CSS (包含自定義動畫與 CRT 特效)
後端服務: Firebase (Firestore, Authentication)
即時通訊: Firestore Realtime Snapshots
圖標庫: Lucide React
